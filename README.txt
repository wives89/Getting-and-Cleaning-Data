The dataset includes the following files:
=============================================

- 'README.txt": shows basic information about the files and how they interact

- 'CodeBook.md': shows information on the variables used in the data, description of how the data and description of data transformations

- 'activitydata.txt': Shows data as described above

- 'datasummary.txt': Summarizes data by mean by activity by person

- 'run_analysis.R': R-Code used to create activitydata.txt' and 'datasummary.txt' from the following original files

Original Data

- 'features.txt': List of all features.

- 'activity_labels.txt': Links the class labels with their activity name.

- 'train/X_train.txt': Training set.

- 'train/y_train.txt': Training labels.

- 'test/X_test.txt': Test set.

- 'test/y_test.txt': Test labels.
